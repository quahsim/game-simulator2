# Assignment: Game Simulator 2
과제대한 질문&답변 링크 : https://github.com/quahsim/game-simulator2/issues/1#issue-2322202428

## 기능
- 회원가입 & 로그인 
- 캐릭터 생성 & 상세조회 & 삭제
- 아이템 생성 & 상세조회 & 목록조회 & 업데이트

## 개발 기술
- 웹 프레임워크인 Express이용
- 데이터베이스 클라우드 서비스인 AWS RDS 이용
- ORM Prisma 사용
- AWS EC2 배포
