# Assignment: Game Simulator 2

## 기능
1. 회원가입 & 로그인 
2. 캐릭터 생성 & 상세조회 & 삭제
3. 아이템 생성 & 상세조회 & 목록조회 & 업데이트

## 개발환경
- 웹 프레임워크인 Express이용
- 데이터베이스 클라우드 서비스인 AWS RDS 이용
- ORM Prisma 사용
- AWS EC2 배포
