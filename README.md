# Assignment: Game Simulator 2

## 기능
1. 회원가입 & 로그인 (JWT TOKEN 발행)
2. 캐릭터 생성 (JWT TOKEN 발행) & 조회 (optional: JWT TOKEN 발행) & 삭제
3. 아이템 생성 & 상세 조회 & 목록조회 & 업데이트

## 개발환경
- Express.js 기반으로 CRUD 사용
- 데이터베이스 RWD 사용
- ORM Prisma 사용
- AWS EC2 배포
